# Fish Value System

## Price Formula

```
Value = math.round(weight * basevalue * starMultiplier * mutationMultiplier)
```

## Star Quality

| Stars | Multiplier |
|-------|------------|
| Dead  | 0.2x       |
| 1     | 0.5x       |
| 2     | 0.75x      |
| 3     | 1.0x       |
